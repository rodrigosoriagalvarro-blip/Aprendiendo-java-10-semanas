# Semana 8: Vaadin — Estructura y Componentes

Mejora de la interfaz de la Agenda Web utilizando componentes estructurales avanzados de Vaadin.

## Estructura de Navegacion
Se ha implementado un MainLayout que hereda de AppLayout para mantener una barra de navegacion constante en todas las vistas del aplicativo.

## Componentes Implementados
- AppLayout: Estructura base con Navbar.
- MenuBar: Menu de navegacion superior.
- TarjetaContacto (Custom): Componente creado en el paquete ui que utiliza Avatar e Iconos.
- Vaadin Icons: Iconos especificos para representar telefono y correo electronico.
- Flexbox: Uso de estilos CSS directos en Java para el diseño en cuadricula de las tarjetas.

## Instrucciones
1. Ejecutar el comando mvn spring-boot:run
2. Acceder a `http://localhost:8080`.
